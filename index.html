PasteBot

It is always a frustrating adventure when a Discord user posts error messages, snippets, configuration files and whole code files either in plain text, in Discords codeblocks or as a file, especially when you are using a mobile device. That's the reason paste services exist. But not many people take the extra effort to help others, and thus, I made PasteBot. A Discord bot automatically reading all sorts of long text and uploading it to a paste service.
